# Mara Voss Runtime Model Folder

Place the final runtime model here:

`mara-voss-rigged.glb`

The app contract expects this public path:

`/assets/models/mara-voss/mara-voss-rigged.glb`

Required companion file:

`mara-voss.manifest.json`

Until the GLB exists, the app will continue using the current image-based operator viewer.
